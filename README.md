# A-Bproject
**Цели**
+ Изучить данные.
+ Привести их в пригодный для анализа формат.
+ Проверить предлагаемые гипотезы.
+ Сделать вывод на основе полученных результатов.

**Вопросы**
+ Тратят ли пользователи с подпиской больше времени на поездки?
+ Верно ли, что среднее расстояние, которое проезжают пользователи с подпиской за одну поездку, не превышает 3130 метров?
+ Будет ли помесячная выручка от пользователей с подпиской по месяцам выше, чем выручка от пользователей без подписки?
+ Подбор методов проверки гипотез для задач по определенным условиям.
